# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - Initial release

- First public release of `@ylakhdar/pretext-breaker`.
- Added a configurable React component for running the text-rendered block breaker easter egg in document-like pages.
- Added trigger controls for the Konami code flow and `?game=true` URL activation.
- Added document brick extraction, game configuration options, and exported renderer/game utilities for custom integrations.
- Added package metadata, bundled stylesheet export, and npm-ready distribution files.
